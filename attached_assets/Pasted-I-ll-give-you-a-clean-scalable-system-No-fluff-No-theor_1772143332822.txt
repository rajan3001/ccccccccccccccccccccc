🧠 PART 5: Schema Markup (Mandatory)

Add JSON-LD schema.

For articles:

{
 "@context": "https://schema.org",
 "@type": "Article",
 "headline": "...",
 "author": {
   "@type": "Organization",
   "name": "LearnPro"
 },
 "datePublished": "...",
 "dateModified": "...",
 "mainEntityOfPage": {
   "@type": "WebPage",
   "@id": "URL"
 }
}

For FAQ section:

{
 "@context": "https://schema.org",
 "@type": "FAQPage",
 "mainEntity": [...]
}

This increases SERP expansion probability.
